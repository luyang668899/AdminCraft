# AdminCraft v2.3.5 发布说明

## 版本信息
- **版本号**: v2.3.5
- **发布日期**: 2026-02-28
- **Git标签**: v2.3.5

## 新功能

1. **完整的后台管理系统框架**
   - 基于ThinkPHP 8.1.1和Vue 3.5.13的企业级后台管理系统
   - 前后端分离架构，支持模块化开发

2. **丰富的功能模块**
   - 认证管理：管理员、角色、权限管理
   - 内容管理：文章、分类、标签管理
   - 电商系统：商品、分类、订单管理
   - CRM系统：客户管理、销售漏斗
   - 项目管理：项目、任务管理
   - 数据分析：仪表盘、报表、数据可视化
   - 第三方集成：支付、短信、邮件、云存储、社交媒体
   - 安全管理：数据回收、敏感数据保护

3. **开发工具**
   - CRUD代码生成：图形化拖拽生成后台增删改查代码
   - 内置WEB终端：支持在后台执行命令
   - 模块化开发：支持插件和模块扩展

4. **性能优化**
   - 常驻内存运行：提升系统性能
   - 按需加载：前端组件和语言包按需加载
   - 缓存机制：提升数据访问速度

5. **安全性**
   - 数据回收：支持恢复误删除数据
   - 敏感数据保护：记录敏感数据操作
   - 权限验证：细粒度的权限控制

## 安装说明

### 环境要求
- **PHP**: 8.1+
- **MySQL**: 5.7+ 或 MariaDB 10.2+
- **Web服务器**: Apache 或 Nginx
- **Node.js**: 16+ (前端开发)
- **Composer**: 2.0+ (PHP依赖管理)

### 安装步骤
1. 克隆代码：`git clone https://github.com/luyang668899/AdminCraft.git`
2. 安装PHP依赖：`composer install`
3. 安装前端依赖：`cd web && npm install`
4. 配置环境：复制 `.env-example` 为 `.env` 并配置数据库信息
5. 执行安装：访问 `http://localhost/AdminCraft/install`
6. 登录系统：默认账号 `admin`，密码 `123456`
